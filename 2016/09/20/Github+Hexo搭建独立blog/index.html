前言

为什么使用Github Pages & Hexo

Github Pages

  1. 搭建简单而且免费；
  2. 可以绑定自己的的域名；
  3. 可以利用很多十分完善且便利的博客框架，比如Hexo,jekyll

Hexo

  1. Hexo 是一个快速、简洁且高效的开源博客框架，用node.js开发，作者是台湾大学生tommy351。
  2. Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

虽然github官方推荐了jekyll，但是再经过比对之后发现hexo要比jekyll配置起来更加的快捷。
所以选择了hexo作为本次搭建博客的框架

准备

既然已经初步了解了Hexo and Github Pages， 下面就开始准备配置所需要的环境。

git (necessary)

  • Mac电脑安装Xcode后会自动安装Git
  • Linux (Ubuntu, Debian)：sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core
  • 下载github的客户端，也会自动配置安装Git
  • 前往git的官网直接下载

学习git推荐廖雪峰大大写的 史上最浅显易懂的Git教程！

node.js (necessary)

  • Mac可以直接用Homebrew安装

关于homebrew的用法可以查阅：homebrew&brew-cask教程

  • nvm安装

  • 直接下载安装程序

Hexo (necessary)

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • git
  • node.js

然后直接
$ npm install -g hexo-cli

github account (necessary)

这个直接去到github网站注册就好了

正式开始

在github创建一个仓库

在Repository name输入yourusername.github.io
yourusername为你自己的用户名全称

然后直接点击Creat Repository即可

设置ssh key

将生成的.pub文件中的内容添加到github的ssh key中

初始化hexo

选择一个建立hexo博客的位置
$ hexo init <folder> #不加入folder选项则默认在当前目录下创建
